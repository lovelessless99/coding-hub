    推薦用 yarn 而不用 npm，快速又好用。

    # 三、使用到的套件

    插件名稱 插件說明
    vuepress-plugin-md-enhance (opens new window) Markdown 的擴充，必裝
    vuepress-plugin-chart (opens new window) 利用 chart.js 的語法畫出美美的圖

    Vuepress Plugin SEO (opens new window) 幫你用 SEO 簡單的優化，讓搜尋排行更前面

    Vuepress Code Copy Plugin (opens new window) 程式碼複製
    vuepress-plugin-comments (opens new window) 留言功能 - 需搭配 Github OAuth 功能得到 ID、Key
    vuepress-plugin-nest (opens new window) 背景動畫網狀圖
    vuepress-plugin-cursor-effects (opens new window) 點下去有星星的特效

    moment (opens new window) SEO
    chart.js (opens new window) 畫圖

    參考網站 (opens new window)

    # 步驟一、先去 flaticon (opens new window) 找你喜歡的 icon

    # 步驟二、去appicon (opens new window) 上傳前一步下載的圖片，網站會生成所有大小的icon

    備註 : icongen - Web and App icons and PNG favicons (opens new window) 這個網站才能幫你留有去背的效果。APP icon 比較美，然後後面的 manifest 檔案就要自己動手修改。

    # 步驟三、解壓縮下載的檔案，並在 .vuepress/public 新增放 icon 資料夾

    我的資料夾是取名成 app_icons，當然你也可以自己改。 之後選用蘋果的資料夾，因為有幫你產生現有的 json 檔，只需要再用下面的程式修改就可以得到 manifest.json

    import json 

if __name__ == "__main__":
        data = json.load(open("Contents.json", 'r'))
        output = {
                "name"            : "Coding Hub"  ,
                "short_name"      : "Coding Hub"  ,
                "start_url"       : "./index.html",
                "display"         : "fullscreen"  ,
                "theme_color"     : "#22979b"     ,
                "background_color": "#000000"     ,
        }
        
        images = []
        
        for image in data['images']:
                item = {}
                item["src"]   = f"./app_icons/{ image['filename'] }" # icon path
                item["sizes"] = image["size"]
                item["type"]  = "image/png"
                images.append(item)
                
        output['icons'] = images
        json.dump(output, open("manifest.json", 'w'), indent=4)
